# TheSixth_Algorithms
Лабораторная работа №6. Вариант 2.<br>Тема: Сортировки.

## №1
Реализовать блочную сортировку **по убыванию** для массива чисел из **N** элементов, используя **6 блоков**. Для сортировки внутри блоков использовать любую сортировку. Исходный массив считывать из текстового файла. Вывести **промежуточный результат**(после деления на блоки) и **конечный результат**(после сортировки каждого блока).
